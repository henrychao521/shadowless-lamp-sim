Phase 0 — 初始研究與分析 完成

2026-05-26 · commit d0a4b9b 初始版本

對現有 3D 介面進行全面分析，識別出四大改善方向：

  • 光學物理：12 顆 SpotLight 配置、IES 光形、平方反比衰減
  • PBR 材質：所有物件使用純色 MeshStandardMaterial，無貼圖導致「塑膠感」
  • 場景構圖：缺少牆面、天花板，SSAO 無空間發揮
  • 後處理管線：Bloom 在 SSAO 前（錯誤順序）

Phase 1 — 後處理修正 + 材質升級 完成

2026-05-26 · commit 15fbe6f
  • 修正後處理順序：Render → SSAO → Bloom → Output
  • Bloom 調整：strength 1.0 → 0.35，radius 0.5 → 0.8（柔光感）
  • 加入 RoomEnvironment IBL：程序生成室內環境光，不需外部 HDR 檔
  • AmbientLight → HemisphereLight：天花板暖光 + 地板冷反射
  • 材質升級 MeshPhysicalMaterial：
    • 地板：clearcoat 0.5（醫療磁磚光澤）
    • 手術台：clearcoat 1.0（拋光不鏽鋼）
    • 覆布：sheen 1.0 + sheenColor（手術布料微光澤）
    • 傷口：clearcoat 0.8, roughness 0.3（濕潤組織）
    • 皮膚：transmission 0.08 + attenuation（次表面散射近似）
    • 燈頭：clearcoat 0.7（烤漆金屬感）
  • 加入手術房：BoxGeometry BackSide 牆面 + 天花板
  • shadow.normalBias = 0.02：消除表面自陰影噪點

Phase 15 — ResizeObserver 精確 Canvas 尺寸 完成

2026-05-26 · 自動 Loop 繼續
  • ResizeObserver 取代 window.resize（simulation3d.js）：
    原本 window.addEventListener('resize', ...) 只在視窗大小改變時觸發，無法偵測「控制面板展開/收合」或「旁邊欄改變寬度」導致的 canvas 容器尺寸變化。 改用 ResizeObserver 監聽 #canvas3d 容器本身，可即時回應任何容器尺寸變化並重設 renderer / composer 大小。 同時加入 if (w === 0 || h === 0) return 防止 div 為零尺寸時觸發無效計算。 不支援 ResizeObserver 的舊瀏覽器回退到 window.resize。

Phase 14 — IEC 閾值線 + 真實模式 Badge 完成

2026-05-26 · 自動 Loop 繼續
  • IEC 60601-2-41 閾值線（simulation.js Chart.js）：
    新增第三條 dataset，固定 y=50 的紅色虛線（borderDash: [3,6]）， 標籤「IEC 60601 最低標準 (50%)」。學生可即時看出照度是否符合國際標準， 遮擋面積過大時紅線在實際照度曲線之上，清楚標示「不合格」。
  • ✨ 真實模式 Badge（index.html + CSS）：
    勾選「切換 3D 真實手術環境模式」後，標題列出現琥珀色徽章「✨ 真實模式」， 採用 badge-appear 動畫（scale 0.8 → 1, opacity 0 → 1）。 取消時隱藏。讓使用者在任何捲動位置都能確認目前渲染模式。

Phase 13 — 效能預連線 + 過捲動行為 完成

2026-05-26 · 自動 Loop 繼續

Phase 11 — 3D 同步 Bug 修正 + 觸感增強 完成

2026-05-26 · 自動 Loop 繼續
  • 🐛 lamp_height 3D 不同步修正（simulation3d.js）：
    原始事件綁定陣列漏掉 'lamp_height'，在 3D tab 上移動燈高滑桿時， updateSimulation3D() 不會被呼叫，燈頭停在上次位置。 現在將 lamp_height 加入綁定清單，確保燈高即時同步更新到 3D 場景。

    滑桿到達 min 或 max 邊界時觸發 18ms 震動，讓使用者不需看螢幕也能感知邊界。 Android Chrome 支援；iOS Safari 無 vibrate API（靜默忽略）。 simulation.js（2D 滑桿）和 simulation3d.js（3D 滑桿）均已加入。
